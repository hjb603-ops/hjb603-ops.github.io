Maths Explanation

Binary is a base-2 number system used by computers, made up of only 0s and 1s.

Decimal is a base-10 system used in everyday counting.

To convert binary to decimal, each digit is multiplied by powers of 2.

Example: 1010₂ = (1×2³) + (0×2²) + (1×2¹) + (0×2⁰) = 10₁₀
